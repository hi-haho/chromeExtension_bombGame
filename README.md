## 크롬 확장 프로그램 설치 방법

크롬 브라우저에서 chrome://extensions/로 이동합니다.

오른쪽 상단의 개발자 모드를 활성화합니다.

압축 해제된 확장 프로그램 로드 버튼을 클릭합니다.

다운로드한 파일을 선택하여 설치합니다.
